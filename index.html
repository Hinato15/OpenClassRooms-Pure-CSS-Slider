Partie 2 : L'interface et les controleurs

L'interface du slider contient les éléments qui vous permette de controler le slider, Elle utilise un systeme qui se sert d'inputs et de labels

Fonctionnement d'une input avec le type "radio"

Vous le savez certainement, vous pouvez dans vos formulaires utiliser des inputs de type radio, pour forcer vos utilisateurs à ne choisir qu'une option parmis plusieurs choix

Si vous cliquez sur une option, celle qui était précedemment séléctionner se déselectionne, l'option sur lequel vous avez cliquez devient alors le choix actif. Vous pouvez vérifier cette explication ci-dessous.

La relation input/label

Cela est interessant mais pour notre module, rien de vraiment utilisable à moins que ... Que ce passe t'il si nous cliquons sur un label au lieu de l'input(pastille) en elle meme.

Vous l'aurez compris, lorsque vous cliquez sur un label, l'input qui lui est associé s'active. Mais il y'a deux chose qui nous interesse plus particulierement :

  • Vous pouvez avoir plusieurs labels qui activent la meme input
  • Vos labels peuvent ce trouvé n'importe ou dans votre fichier
